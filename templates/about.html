About XMEME

XMEME is a web based application which is primarly used to store the meme, fetch the meme, and update a particular meme by url and caption. All the operation can be performed using RESTFUL API structure.

Technologies Used:

  • Flask
  • MongoDB
  • HTML
  • Python
  • Heroku
  • Bootstrap
